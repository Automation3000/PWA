Quick Start Guide

Welcome to the TABREED Automation App v2.0! Here's a quick overview of the new features and how to use them.

🌟 What's New?

✨ Dark Mode

Click the moon icon in the header (top right) to switch between light and dark themes. Your preference is automatically saved.

🔄 Better Offline Support

Changes work even without internet. The system will automatically sync when you're back online using a smarter retry system.

📱 Daily Usage

Making Changes

  • Search: Use the search bar at the top or filter by location.
  • Update: Click "Update" button on any row, fill details, and click "Save Changes".
  • Add: Click "Add Entry" button, fill details, and click "Save Changes".

Going Offline

  • 🔴 Red "Offline" indicator shows no internet.
  • 📝 You can still make changes (saved locally).
  • ⚠️ A badge shows pending changes waiting to sync.

📊 Viewing Data

  • View History: See all past changes with timestamps.
  • Stats: View total items, locations, and expiring items.
  • Expiry Due: See items expiring soon. Click on the Expiry column header to toggle between actual date and days remaining.

💡 Pro Tips

  • Batch Updates: Make all edits offline if possible, and let them sync together when you connect.
  • Regular Backups: Click "Download PDF" periodically to keep archives.
